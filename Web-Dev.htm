    1. Html BoilerPlate-Basic template of Html. 
      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>File Title</title>
</head>

</body>
</html>
    2. Adding CSS-Liking CSS file to Html file. 
      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    Use link element to add CSS file
    <link rel="stylesheet" href="Your file.css">
    <title>My Title</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
    3. Adding JS-Liking JS file to Html file. 
      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="Your file.css">
    Use Script source element to add Js file
    <script src="Your file.js"></script>
    <title>My Title</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

How to install MongoDB

MongoDB is a source-available cross-platform document-oriented database program. Classified as a NoSQL database program, MongoDB uses JSON-like documents with optional schemas.

Click the below link and it will redirect you to MongoDB community download page.

Go to MongDB site

After clicking the link you will be redirected to this page, here you have to click on MongoDB Community Server:

Then you have to just click on the download button:

After downloading you have to run this setup and this screen will come:

After that you will come to this screen, then you have to click on complete and then next with default settings:

Then you have to make sure that you have check the Install MongDB compass and then click on Install:

MongoDB compass is a GUI that helps you manage you database without write a single line of command.

After sometime MongoDB compass install and launch in your system:

After that you have to create data folder in you C drive:

You have to go where you have install MongoDB and then server/5.0/bin:

Here is the default path

C:\Program Files\MongoDB\Server\5.0\bin

Then you have to search Advanced System Setting

Then you have to click on enviorment variable:

Then you have to click path in upper panel and click edit:

After that you have to click on New and then paste your path and then just click ok:

After that open powershell or command prompt and run this command:

mongo

If you get any error try this method again.if still you get error you can message me

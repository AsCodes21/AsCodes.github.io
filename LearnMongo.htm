Learn MongoDB commands with me

If you want to know how to install MongoDB you can go to my blog

1. Database Commands

View all databases

show dbs

Create new or switch to db

use db

View current db

db

Delete currently using database

db.dropDatabse()

2. Collection Commands

Show all collection:

show collections

Create a collection

db.createCollection('collection_name')

Delete collection

db.collection_name.drop()

3. Row Commands

Show all rows:

db.collection_name.find()

Show all rows in good view:

db.collection_name.find().pretty()

Show only limited rows:

 db.collection_name.find().limit(2)

Count number of outputs:

db.collection_name.count()

Add a data json:

db.collection_name.insert({
      OBJECT INFO     
      })

Add more than 1 json:

db.colllection_name.([{OBJECT-1 INFO},{OBJECT-2 INFO}])

Remove a row:

db.collection_name.remove({search query})

4. Some other commands

Find rows that match with this criteria:

db.collection_name.find({"key","value"})

Find only first row that match with this criteria:

db.collection_name.findOne({"key","value"})

Edit a field : 

db.collection_name.update({search query},{
          "key":"value"
      })

Add a new obejct if field doesnt match :

db.collection_name.update({search query},{
          "key":"value"
      },{upsert:true})

Rename fields name:

db.collection_name.update({"key":"value"},{
    $rename:{
        "key":"value"
    }})

Less than/Greater than/ Less than or Eq/Greater than or Eq Operator:

db.collection_name.find({"age":{$lt:90}})
db.collection_name.find({"age":{$lte:90}})
db.collection_name.find({"age":{$gt:90}})
db.collection_name.find({"age":{$gte:90}})
